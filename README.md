## 📋 О проекте

**To-do list** — это современное веб-приложение для эффективного управления личными задачами. Проект разработан с использованием лучших практик Django и представляет собой полноценную систему с авторизацией, кастомизацией задач через теги и интуитивно понятным интерфейсом.


## 🚀 Ключевые возможности

### 👤 **Пользователи**
- 🔐 Регистрация и безопасный вход в личный кабинет
- 👥 Кастомная модель пользователя для расширенной функциональности
- 🔒 Разграничение доступа: каждый пользователь видит только свои задачи

### ✅ **Управление задачами**
- ➕ Создание новых задач с заголовком и описанием
- ✏️ Редактирование существующих задач
- 🗑️ Удаление ненужных задач
- ✔️ Маркировка выполненных дел

### 🏷️ **Теги**
- 📌 Добавление тегов к задачам для удобной категоризации
- 🔎 Фильтрация задач по тегам
- ✂️ Возможность множественного выбора тегов

### 🎨 **Интерфейс**
- 📱 Адаптивный дизайн на Bootstrap 5
- 🖥️ Интуитивно понятный интерфейс
- ⚡ Быстрый доступ ко всем функциям
- 💬 Система уведомлений об успешных действиях

## 🛠️ Технологический стек

| Компонент | Технология         |
|-----------|--------------------|
| **Backend** | Python 3.10+, Django |
| **База данных** | PostgreSQL 17      |
| **Аутентификация** | Кастомная модель User |
| **Frontend** | HTML5, Bootstrap 5 |
| **Шаблонизация** | Django Templates   |
| **Тестирование** | Pytest, BeautifulSoup4 |

## 📁 Структура проекта

| Приложение | Описание                           | Ключевые модели |
|:-----------|:-----------------------------------|:----------------|
| **core** | Кастомные страницы для ошибок      | -               |
| **homepage** | 🏠 Главная страница с приветствием | -               |
| **task_list** | ✅ Управление задачами              | Task, Tag       |
| **user_profile** | 👤 Профиль пользователя            | UserProfile         |
| **registration** | 🔐 Аутентификация и регистрация    | -               |

## 🗄️ База данных

Проект использует **PostgreSQL** для надежного хранения данных:
- Пользователи хранятся в кастомной модели User
- Задачи привязаны к конкретным пользователям
- Теги имеют связь многие-ко-многим с задачами
- Все связи оптимизированы для быстрых запросов

## 🧪 Тестирование

В проекте реализовано комплексное тестирование с использованием **Pytest**.\
Директория pytest_tests/ содержит все тесты проекта, организованные по приложениям.\
В корне проекта хранится конфигурационный файл **pytest.ini**

## 📦 Управление зависимостями с Poetry

Проект использует **[Poetry](https://python-poetry.org/)** для управления зависимостями и виртуальным окружением. Poetry — это современный инструмент для управления пакетами в Python, который обеспечивает надежное и воспроизводимое управление зависимостями.

### 📋 Файлы Poetry в проекте

| Файл | Назначение                                                                                                  |
|:----:|:------------------------------------------------------------------------------------------------------------|
| **pyproject.toml** | Основной файл конфигурации, содержащий метаданные проекта и список всех зависимостей с допустимыми версиями |
| **poetry.lock** | Автоматически генерируемый файл, фиксирующий точные версии всех установленных пакетов                       |
